This repository contains my lab projects on networking and cybersecurity, all built in GNS3. Each project shows how I design, configure, secure, and test networks on real Cisco equipment, using penetration testing and monitoring tools.

I built this portfolio to demonstrate my hands‑on skills – not just theory. All labs are fully reproducible – just follow my step‑by‑step guides. 🛠️

Repository structure:

📂 basics – networking fundamentals: VLAN, Trunk, Etherchannel, OSPF, HSRP, GRE
🔐 security – network protection: ACL, Port Security, DHCP Snooping, SSH
⚔️ protocol-attacks – layer‑2/3 attacks: ARP Spoofing, DNS Spoofing, VLAN Hopping, DHCP Starvation
🎯 pentest-lab – full attack cycle: reconnaissance, exploitation, post‑exploitation
🏢 enterprise – corporate networks: DMZ, VPN, high availability
📊 soc-integration – monitoring: IDS/IPS (Snort), log collection (ELK), incident analysis

Each lab folder contains:
📘 LAB_GUIDE.md – step‑by‑step commands and explanations
⚙️ configs – final configurations of all devices
📸 screenshots – topology, setup, and verification results

Technologies I use:
🖥️ GNS3, Cisco IOS, Kali Linux
🔍 Wireshark, Metasploit, Nmap
🛡️ Snort, ELK, Wazuh, pfSense

Important: ⚠️ All projects are built in isolated lab environments and are intended for educational purposes only. I am not responsible for any misuse of these materials in real networks.
